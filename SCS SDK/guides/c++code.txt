Title: C++ Code

    Some information to handle the cpp files in the telemetry.

About: #pragma region

    Like you can see in the file <scs_config_handlers.cpp> the visual studio keyword 
    ======= C++ =======
    #pragma region RegionName
     ...
    #pragma endregion Comment
    ===================
    is used. With the help of this lines, you can outline the region with a click on the `-`. 
    In other IDE's or editors it may not give you that possibility, but it still should help to understand the region.
